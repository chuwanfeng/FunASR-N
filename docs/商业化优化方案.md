# FunASR-N 商业化优化方案

## 项目概述

FunASR-N 是一个基于 FastAPI + FunASR 的语音识别字幕生成器，针对 CPU 优化（目标 i5-12400），支持三引擎：Paraformer-zh、SenseVoiceSmall、Qwen3-ASR-0.6B（OpenVINO）。

---

## 优化方向一：越用越准确（用户反馈学习）

### 现状
- 前端无反馈 UI，后端无反馈 API
- 训练脚本存在但无闭环

### 已实现

#### 1. 反馈存储模块 (`feedback_store.py`)
```python
class FeedbackStore:
    - save_feedback()      # 保存用户修正
    - get_feedback()       # 查询反馈记录
    - extract_hotwords()   # 提取高频修正词
    - update_config()      # 自动更新热词配置
```

#### 2. 后端 API (`main.py`)
- `POST /feedback` - 提交反馈
- `GET /feedback` - 查询反馈
- `POST /feedback/extract-hotwords` - 提取热词

#### 3. 前端标注 UI
- 行内编辑字幕文本
- 修改标记（绿色边框）
- 单条/批量提交反馈
- 撤销/重做（Ctrl+Z/Y）
- 实时预览修改后的 SRT

### 使用流程
```
上传音频 → 识别 → 切换到"时间轴片段" → 点击文本修改 → 提交反馈 → 后端存储 → 定期提取热词 → 更新配置
```

## 优化方向二：时间轴准确性

### 现状
- 词级时间戳 + 标点拆分
- 三级降级策略

### 已实现

#### 1. VAD 静音检测
```python
USE_VAD = True
VAD_MIN_SILENCE_DURATION = 0.3  # 静音超 0.3s 则切分
```

#### 2. 代码修复
- **P4**: `format_time` 毫秒计算修复（进位丢失）
- **P5**: 降级方案时长分配不越界
- **P6**: 全局单例线程安全（双检锁）

#### 3. 三级降级策略
```
1. 词级时间戳精确匹配
2. 有时间戳但词数不匹配 → 回退到整段
3. 无时间戳 → 按标点拆分，比例分配时长
4. 无标点且 duration > 12s → 固定切分
```

## 配置参数

```python
# 核心配置
ASR_ENGINE = "paraformer-zh"
DEVICE = "openvino"
OPENVINO_DEVICE = "GPU"

# VAD
USE_VAD = True
VAD_MIN_SILENCE_DURATION = 0.3

# 字幕控制
SRT_MAX_SEGMENT_DURATION = 8.0
SRT_MIN_SEGMENT_DURATION = 1.0
ASR_SEGMENT_DURATION = 15.0

# 服务
SERVER_HOST = "127.0.0.1"
SERVER_PORT = 8889
```
